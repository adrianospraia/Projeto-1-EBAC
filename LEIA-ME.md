# Projeto 1 EBAC
Primeiro projeto hospedado no GitHub do curso de analista de dados da EBAC

Projeto do módulo de Git do curso EBAC

Instruções de execução da calculadora em Python: 

1 - Na página principal do "Projeto-1-EBAC" clique na opção "<Code>" em destaque verde

2 - Surgirá uma sub-lista com algumas opções, clique onde diz "Download ZIP code"

3 - Extraia o arquivo baixado no seu dispositivo

4 - Entre na pasta "projetocalculadora"

5 - Dê um duplo clique no arquivo "projetocalculadora"
